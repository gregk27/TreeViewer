# TreeViewer
Simple Tree viewer following the output format used by supplied code from ELEC278.

The following input:
`{(5,e),{(3,c),{},{(1,a),{},{}}},{(7,g),{},{}}}`
Would be rendered as:

![Image](https://i.imgur.com/WQqA9Vg.png)

Rendering is done by [Treant](https://fperucic.github.io/treant-js/)
